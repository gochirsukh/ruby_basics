### Example
```
irb(main):001:0> x = 1
irb(main):002:0> x == 1
=> true
irb(main):003:0> x == 2
=> false
```

### And operator && 

* The both condition must be true 

```
irb(main):008:0> x > 0 && x < 100
=> true
```
### Or operator || 

* Only of the of the condition can be true

```
irb(main):010:0> x >= 100 || x <= 50
=> true
```
