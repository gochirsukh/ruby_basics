## Itererators 

* Apply procedure repeatiatly 
* To perform code on each item in a set

```
i = 5 

i.times do 
    puts "Countdown: #{i}"
    i -= 1
end 

puts "Blast off!"
```

### Downto

```
5.downto(1) { puts "Hello" }
```


### Each

```
(1..5).each { puts "Hello" }
```

### Times 

```
5.times { puts "hello"}
```

### Upto

```
1.upto(5) { puts "Hello" }
```
