## Unless 

* It's negation of boolean 
* Oppositive of boolean 
* It's easier to read

```
unless boolean
  # code 
end
```

* Unless is same as 

```
if !boolean
    # ...
end 
```

* Example-1 

```
unless array.empty?
    # ...
end
```

* Example-2

```
unless search_result.nil?
    # ...
end 
```

* Example-3

```
if product.visible?
    #...
end 
```

* Example-4

```
unless product.sold_out?
    #...
end
```
